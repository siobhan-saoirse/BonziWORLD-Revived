BonziWORLD Revived

Leading the industry in gorilla-based chat clients.

What the hell is this?

BonziWORLD is a node.js and socket.io based chat client featuring everyone's most hated loved purple gorilla. The client also uses espeak.js to provide text-to-speech voices. Not exactly the original BonziBUDDY voice, but hey, it works.

This website is a parody of BonziBUDDY and has no affiliation to it.

How do I use it?

  1. Enter a nickname (under 25 characters). If you don't enter one, you'll be named "Anonymous". (we r legion du nut 4 get)
  2. Optionally, enter a room ID. If you want to join someone else, their room ID will be in the bottom-right corner of the screen.
    • If you enter a room ID that doesn't exist, you will be placed in a private room which can only be joined by people you share the ID with.
  3. Be a BonziBUDDY.

What's new about this?

  • The chat logger is better and you can now reply to messages.
    • You can also resize it. Huge, I know.
    • You can clear the chat with /clear.
  • You can mute other people. (This literally wasn't an option before)
    • You can middle click on people to mute them.

That's the jist of it, barring some stupid shit

Are there any rules?

Obviously no illegal shit. On top of that, there are a few limitations:

  • All strings are sanitized to get rid of any malicious HTML/JS/CSS. Don't even try it motherfucker.
  • There is a 2500 character limit in public rooms and a 5000 limit in private rooms.
    • Yes, the Navy Seals copypasta fits.
  • There is a 25 character limit on names.
  • Discrimination is not allowed here. Homophobia is also a bannable offense in Erik's competing server bonzi.gay.
  • Anyone who sends illegal content (both irl and fictional child sexual abuse material, and shock media) will be reported to the police.
  • Any kind of sexual content is not allowed. There are actual kids using this website.

Remember that people can play YouTube videos whenever they want. Keep your volume down, headphone users. Chat rooms are for the most part unmoderated. If shit starts getting spammy, homophobic or abusive, I might start using my banhammer.
